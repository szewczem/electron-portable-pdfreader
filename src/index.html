help section

New window

To open pdf in a new window, press Shift + left mouse click on the selected pdf in the MENU.

PDF Viewer

Documents are displayed by PDF Viewer. The toolbar at the top of the window allows you to select a page and zoom, adjust the width/height of the window, rotate pages, save, print, and set the view of two pages of the document. Documents with a table of contents allow you to select a specific chapter from the document table.

Download and print function

All pdf contained in the application can be downloaded and printed. The saving process, depending on the size of the saved file, can take from a few to a dozen or so seconds.

It is recommended to save the pdf to disk using the "Download" function.
